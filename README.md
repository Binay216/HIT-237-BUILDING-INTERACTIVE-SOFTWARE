# HIT237 - Building Interactive Software

## NT Remote Housing Repair System

This project is a Django-based web application designed to manage and track repair requests for remote housing communities in the Northern Territory. The system aims to improve efficiency, transparency, and communication between tenants, service providers, and administrators.

## Project Overview

The application allows users to submit repair requests, monitor progress, and manage maintenance workflows. It provides a structured system for handling repair operations in remote areas where accessibility and coordination can be challenging.

## Features

* Submit and manage repair requests
* Track repair status and updates
* Organise maintenance workflows
* Centralised system for housing repair management

## Technologies Used

* Python
* Django Framework
* HTML, CSS
* SQLite (default database)

## How to Run the Project

1. Install dependencies:

   ```
   pip install -r requirements.txt
   ```
2. Navigate to project directory:

   ```
   cd HIT-237-BUILDING-INTERACTIVE-SOFTWARE
   ```
3. Run the development server:

   ```
   python manage.py runserver
   ```
4. Open in browser:

   ```
   http://127.0.0.1:8000/
   ```
